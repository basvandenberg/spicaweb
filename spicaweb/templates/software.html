Software

spica

The python package spica provides two command-line tools to perform protein feature extraction (featext.py) and classification tasks (classification.py). The website uses these tools at the back-end for feature calculation and classification. The source code is available on github. Please consult the package documentation for more information about installation and use of this software.

spicaweb

The python package spicaweb contains the code to run this website. Source code is accessible on github.

Software

spice

Python package spice provides two command-line tools to perform protein feature extraction (featext.py) and classification tasks (classification.py), which run at the back-end of this website. Source code is available on:

www.github.com/basvandenberg/spice

Please consult the package documentation for more information about installation and use of the software.

spiceweb

The spiceweb software contains the source code of this website and is accessible on:

www.github.com/basvandenberg/spiceweb

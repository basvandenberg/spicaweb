Software

spice

Python package spice provides three command-line tools that run at the back-end of this website to perform: protein feature extraction (featext.py), classifier construction and testing (classification.py), and running a trained classifier on another data set (classify.py). Source code is available on:

www.github.com/basvandenberg/spice

Please consult the package documentation for more information about software installation and usage.

spiceweb

The spiceweb software is used to run this website, the source code is accessible on:

www.github.com/basvandenberg/spiceweb

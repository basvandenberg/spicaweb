About

SPiCE provides easy access to visualization and classification methods for a set protein sequences. It can be used to calculate sequence-based protein features, visualize feature data, and construct protein classifiers.

SPiCE has been developed as part of my PhD in the Delft Bioinformatics Lab at Delft University of Technology. The software was initialy used for the construction of a classifier that predicts successful high-level secretion of proteins in Asperigillus niger (1). To make feature exploration and classifier construction methods generally accessible, we developed the SPiCE website as a user-friendly graphical user interface to (most of) that software.

A registered account is required to use the website, this to provide a protected workspace for each user. You need te register with a valid email address and a password. A verification link will be send to you in order to activate your account.

The website performance best in Google Chrome. Both Firefox and Opera have good support, however, in both browsers there seems to be an isseu with showing the t-statistics table. Please avoid using Internet Explorer (espessially the older versions), not all functionalities work in this browser. Please contact me in case of any other issues.

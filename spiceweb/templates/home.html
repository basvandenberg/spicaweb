Welcome to the SPiCe website

SPiCe is a web application that can be used to explore global properties of a set of protein sequences. After uploading a fasta file with protein sequences, the application can be used to calculate a range of sequence-based features. The web application can also be used to train and test sequence-based classifiers if protein labels are provided.
